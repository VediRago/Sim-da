# DA Simulation Lab

Public GitHub Pages repository for small standalone simulation tools used to test DA game systems.

## Structure

```text
index.html
tools/
  dicetest/
    index.html
    reports/
```

Each simulator should test one isolated design question and export Markdown reports for later AI discussion.

Future tools, such as village growth, should use the same pattern:

```text
tools/village-growth/index.html
tools/village-growth/reports/
```
